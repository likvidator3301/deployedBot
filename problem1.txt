Необходимо создать бота, выдающий новости в сфере IT, материалы по обучению по определенным тегам.
Материал планируется брать из IT-сообществ vk.
Для пользователей уже пользовавшихся ботом, он будет выдавать новости по тегам, которые он запрашивал, рекомендовать обучающий материал по смежным темам.


Бот общается с пользователем через консоль. Пользователю необходимо представиться, чтобы бот отличал его. Бот будет хранить его имя, теги которые запрашивал пользователь. Через n-ое время после последнего запроса пользователя, бот будет либо искать новости по этим тегам, либо выдавать какой-то обучающий материал по этим тегам(в консоль).
